Unfurl — тестовые страницы

Сохраните каждый раздел этого файла как отдельный HTML и разместите на тестовом домене/сервере. Затем используйте ссылки ниже в мессенджерах/ботах/социальных сетях, чтобы увидеть, как сервисы их превьюируют.

Примеры страниц

OG — большая картинка (1200×630) — og-large.html OG — маленькая картинка (400×300) — og-small.html Twitter Card — summary_large_image — twitter-card.html OG video — видео-превью — video-card.html No image — только текст — noimage.html

Инструкция

  1. Скопируйте секцию (например ) в отдельный файл с указанным именем.
  2. Разместите файлы на вашем сервере (или используйте локальный сервер — e.g. python -m http.server).
  3. Откройте ссылку в мессенджере/боте/социальной сети/инструменте для проверки карт-превью (например Facebook Sharing Debugger, Telegram, Slack, Twitter Card Validator).
  4. Если превью не обновилось, используйте инструмент платформы для очистки кеша (Sharing Debugger) или измените URL (параметр ?v=1) чтобы принудить повторное сканирование.
